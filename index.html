Current status

The developers of the land have put in a Section 73 notification for the development of the land south of Church Fenton. It is listed on the Selby Planning Portal as application number ZG2024/0187/S73

A Section 73 submission is to ALTER an existing application. The alterations in this case are to the layout and street scape.

There is a search button on the link above - if you put the application reference into that Search bar, and press Search - it will bring up the application. Alternatively, you can use this link.

Alternatively, you can simply email planningcomments@selby.gov.uk and express your objection to this application
